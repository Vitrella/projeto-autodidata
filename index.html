HTML

O HTML é uma linguagem de tags, serve principalmente para mapear e dar caracteristicas aos elementos que estarão presentes no site.

HEAD

Na HEAD adicionamos informações que serão lidas pelos navegador, mas não necessáriamente aparecerão na página


META TAGS

As META TAGS ficam na "head" do código, são responsáveis para enviar algumas informações para o servidor, veja abaixo alguma das META TAGS mais usadas:

<#meta charset="UTF-8">
<#meta name="viewport" content="width=device-width, initial-scale=1.0">
<#meta name="author" content="BOLINHO">
<#meta name="description" content="Site de teste">
<#meta name="keywords" content="teste, teste01, teste02">

(As "#" antes das tags "meta" não são usadas no código em si, estão sendo usadas apenas para o código aparecer no site)


TITLE (NOME DA ABA)

Na HEAD também escolhemos o nome da página, aquele nome que vai aparecer na aba que estamos acessando

<#title>NOME DA ABA<#/title>


Títulos de texto

Os titúlos são os "h1,h2,h3,h4,h5,h6", são responsáveis pela hierarquia dos títulos, sendo do h1 o maior e principal, até o h6, o menor e menos "importante"


Devemos usar apenas um H1 por página, mas podemos usar depois quantos H2 quiseros. (Essas regras são usadas para rankear melhor o site no google)


Marcadores de texto

Os marcadores servem para destacar determinadas partes de um texto, e cada um tem sua função:


TEXTO PARA TESTAR MARCADORES <#i> - Itálico


TEXTO PARA TESTAR MARCADORES <#b> - Negrito


TEXTO PARA TESTAR MARCADORES <#mark> - Marca texto


TEXTO PARA TESTAR MARCADORES <#strong> - Aparenta como um negrito, mas além disso, diz para o navegador que o texto é importante e relevante no geral


TEXTO PARA TESTAR MARCADORES <#sub> - O texto vai para baixo


TEXTO PARA TESTAR MARCADORES <#sup> - O texto vai para cima


TEXTO PARA TESTAR MARCADORES <#del> - Risca o texto no meio (bastante usado em descontos)


TEXTO PARA TESTAR MARCADORES <#u> - Sublinha o texto



Elementos de citação

Os Elementos de citação são usados para demarcar, falas, obras (como filmes, livros, peças e etc), endereços e mais:


Ontem assisti Titanic <#cite> - Cita obras


Então Eduarda disse: Oie sheldinho! - Cita falas adicionando ""


Sheldon mora na rua américo brasiliense, 113 - Cita um endereço

Fazendo comentátios

O Comentário em um código é um texto que está ali dentro do arquivo, porem não é cosiderado código então não é lido pelo servidor, serve apenas para fazermos anotações em nossos códigos para nos orientarmos.

<#!--AQUI DENTRO FICA O COMENTÁRIO-->
